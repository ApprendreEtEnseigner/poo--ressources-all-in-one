Les variables en POO

Quoi ?

Imaginez une boîte qui peut contenir une valeur.

Cette boîte est appelée variable en POO.

Elle peut contenir différents types de valeurs.

Les variables en POO

Qui ?

Les programmeurs utilisent les variables en POO.

Cela aide à stocker et manipuler les données.

Même un robot dans un jeu vidéo utilise des variables pour suivre sa santé.

Les variables en POO

Où ?

Partout dans la programmation.

Dans les jeux, les applications, les sites web, etc.



Les variables en POO

Quand ?

Chaque fois qu’on a besoin de stocker une information.

Quand on note un score dans un jeu.

Quand on enregistre le nom d’un utilisateur.

Les variables en POO

Comment ?

On déclare une variable, comme on nomme une boîte.

On lui attribue une valeur, comme on met quelque chose dans la boîte.

On peut ensuite utiliser cette valeur, comme on utilise ce qui est dans la boîte.

Les variables en POO

Combien ?

Il existe plusieurs types de variables, comme...

...des boîtes de différentes tailles et formes.

Certaines peuvent contenir des nombres, d’autres des textes, et...

Les variables en POO

Combien ?

...certaines peuvent même contenir d’autres boîtes!

Par exemple, il y a des variables entières, des variables à...

...virgule flottante, des variables de chaînes de caractères, etc.

Les variables en POO

Pourquoi ?

Pour stocker et manipuler des informations.

Pour se souvenir des informations importantes.

Parce que sans elles, nos programmes seraient...

Les variables en POO

Pourquoi ?
